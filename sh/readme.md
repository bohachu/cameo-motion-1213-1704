# 安裝
001 cd 家目錄
git clone <本目錄> 

002 將private key和public key放置到專案目錄/secrets 目錄 下面

003 打包:將網站發布到wwwhtml目錄中
cd 專案目錄/sh 
./compile.sh

004 安裝主程式
cd 專案目錄/sh 
執行 ./install.sh
